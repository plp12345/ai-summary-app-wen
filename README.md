## Overview

Agentic Coding agents allows users to generate thousands of lines of code and build complex features without fully understanding how the code works.  While AI can easily complete the first 70% of a platform, what truly differentiates us is our understanding of the AI-generated code or solutions, which may appear correct on the surface but can contain inefficient or flawed logic underneath. Our ability to debug and fix these problems ourselves, especially in the final 30% where AI struggles, is what sets us apart. In the age of AI, it is still important to learn the foundational skills of building applications step-by-step. 

In this tutorial, we will 
 - Build an AI Summary App using Next.js, Tailwind CSS, and Supabase. The app will allow users to upload documents, preview their content, and generate AI-powered summaries. 
 - You will also learn how to develop your own customized tutorials using AI to learn new technology effectively.

## Learning objectives

- Develop a practical understanding of building a modern web app with Next.js, Tailwind CSS, and Supabase.
- Gain experience working collaboratively with AI tools, learning when and how to trust, verify, and improve AI-generated code.

- Learn how to develop customized learning contents as a method of consolidating knowledge learnt and develop valuable resource to revisit later to improve understanding.

### Important Notes/Tips
- Don't just let the AI agent complete all the step. If you do so, you will miss the opportunity to learn and understand the concepts behind. You should try the steps one by one to make sure you understand the foundation. 
- The AI agent may not always generate perfect code or documentation. 
    - Always review and test the generated content to ensure it meets your requirements. 
    - Update your tutorial when the tutorial steps are not accurate or miss important explanation.
- Debug and Refactoring
    - When encountering bugs you want the AI agent to help fix, provide relevant information such as error messages, terminal logs, browser console logs, and platform logs (e.g., Vercel).
    - While implementing your app, you can use AI to help refactor and modularize the code into multiple files to improve its reusability and maintainability.
